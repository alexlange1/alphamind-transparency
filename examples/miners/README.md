# Miner examples

- Emit one-shot emissions:

```bash
python -m subnet.miner.loop --emit-emissions-once
```

- Emit one-shot prices:

```bash
python -m subnet.miner.loop --emit-prices-once
```

Use hotkey signatures where possible.
